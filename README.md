# הרפתקת המתמטיקה 🎈

משחק מתמטיקה לנויה (7, כיתה ב') ולאלין (4) — לשימוש משפחתי, על טאבלט סמסונג.

- **נויה** — 60 שלבים, 10 עולמות: חיבור וחיסור עד 100, לוח הכפל, חילוק (בלי שארית), ובעיות מילוליות מנוקדות — עם לוח מספרים על המסך.
- **אלין** — 18 שלבים, 3 עולמות: הכל בתמונות (ספירה, השוואות, חיבור חזותי), כל שאלה מוקראת בקול.
- כוכבים על כל שלב, אלבום מדבקות, ודמות מלווה שגדלה עם ההתקדמות.
- **חנות** 🛍️: מטבעות (תשובה נכונה = מטבע, +5 לכל כוכב בסוף סבב) קונים אביזרים, צבעים, רקעים, ביצי הפתעה עם פריטים נדירים, ודמויות חדשות (חד-קרן/פנדה/כלבלב, 100 🪙). הכוכבים לא יורדים אף פעם — רק המטבעות מוצאים. מעבר חופשי בין דמויות שבבעלות במסך הדמות.
- עובד **אופליין לגמרי** אחרי ההתקנה. כל ההתקדמות נשמרת על הטאבלט עצמו. בלי חשבונות, בלי שרת, בלי איסוף נתונים.

## הרצה מקומית (לפיתוח)

```
python -m http.server 8765 --directory math-adventure
```

ואז לפתוח `http://localhost:8765` בדפדפן. (אי אפשר לפתוח ישירות כקובץ — נדרש שרת בגלל מודולים ו-service worker.)

בדיקות הגנרטור: לפתוח `http://localhost:8765/tests.html` — מריץ 1,000 דגימות לכל שלב ומוודא שכל האילוצים מתקיימים.

## העלאה לאינטרנט (פעם אחת, להתקנה על הטאבלט)

1. ליצור repo ב-GitHub ולדחוף את תיקיית `math-adventure`.
2. Settings → Pages → Deploy from branch → main → root (או `/math-adventure` אם זה תת-תיקייה).
3. לחכות דקה — הכתובת תהיה `https://<שם-משתמש>.github.io/<שם-הריפו>/`.

## התקנה על הטאבלט (סמסונג)

1. **לוודא קול עברי**: הגדרות → ניהול כללי → טקסט לדיבור → שירותי הדיבור של Google → לוודא שיש קול עברית (להוריד בזמן שיש אינטרנט).
2. לפתוח את הכתובת ב-**Chrome** בטאבלט.
3. לשחק סבב אחד כדי לוודא שהקול והאימוג'י נראים טוב.
4. תפריט ⋮ → **"הוספה למסך הבית"** / "התקנת אפליקציה".
5. לבדוק: מצב טיסה → לפתוח מהאייקון → הכל אמור לעבוד.

## עדכון גרסה

אחרי כל שינוי בקבצים: להעלות את מספר הגרסה ב-`sw.js` (שורה ראשונה: `math-adventure-v1` → `v2`), לדחוף ל-GitHub, ולפתוח את האפליקציה פעמיים בטאבלט כשיש אינטרנט (הפתיחה הראשונה מורידה, השנייה מציגה).

**חשוב**: לא למחוק נתוני אתר/דפדפן של הכתובת בטאבלט — שם נשמרת כל ההתקדמות של הבנות.

## מבנה הקוד

```
js/curriculum/   הגדרות השלבים (התוכן) — נויה, אלין
js/engine/       יצירת תרגילים אקראית + כוכבים/מדבקות/דמות
js/screens/      המסכים: פרופילים, מפה, תרגיל, פרסים, דמות
js/ui/           רכיבים: קבוצות אימוג'י, לוח מספרים, SVG של הדמות, קונפטי
sw.js            service worker — עבודה אופליין (להעלות גרסה בכל עדכון!)
```
